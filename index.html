Full Stack Web Application

Sports with Strangers

Single-page web application to facilitate meet-ups for sporting events. Developed in JavaScript with React, Redux, and jQuery on the frontend. The backend utilizes Ruby on Rails coupled with a postgreSQL database, and jBuilder to architect API responses. Developed as a solo project.

JavaScript Web Demonstration

Pixelary

Demo of familiar image processing filters presented in the form of pictionary, written in JavaScript. Image filtering algorithms were adapted for use with HTML5 DOM elements. Built primarily in vanillaJS with jQuery for DOM text manipulation. Developed as a solo project.

Android Mobile Application

Platform for sharing food photography and finding new recipes. Developed in Android Studio using Java and XML. Handles calls to Food2Fork API asynchronously and parses JSON to create new Bitmaps. Developed in collaboration with Kathy Luo and Michael Mach.

Software and Technical Projects

Sports with Strangers

Sports with Strangers is an application taking inspiration from Tea with Strangers, which gives users a platform to join local meet-ups to watch sporting events, find new fans in their home city, or host meet-ups themselves.

Pixelary

Pixelary is a recreation of pictionary meant to demonstrate image processing in an engaging way. The script extracts an array of RGBA values from an image and rebuilds it pixel-by-pixel with a filter applied. Algorithms for each image filter were adapted into JavaScript and applied in increments to achieve an animation effect.
