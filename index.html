Full Stack Web Application

About Sports With Strangers

Sports with Strangers is a single-page web application to facilitate meet-ups for sporting events. Developed in JavaScript with React, Redux, and jQuery on the frontend. The backend utilizes Ruby on Rails coupled with a postgreSQL database, and jBuilder to architect API responses. The REST API makes AJAX requests to the server and receives the data in JSON to update Redux' application store. React-Router was used to manage location hash and protect restricted routes.

JavaScript Web Demonstration

About Pixelary

Pixelary is meant to demonstrate image processing in an engaging way. The script extracts an array of RGBA values from an image and rebuilds it pixel-by-pixel with a filter applied. Algorithms for each image filter were adapted into JavaScript and applied in increments to achieve an animation effect. Multiple asynchronous event loops control the timer, animation, and user input. All game logic and image processing was written using vanilla JS, with jQuery being used for easier DOM text manipulation. Developed as a solo project.

Android Mobile Application

About CheffyPix

The app is a platform for sharing food photography and finding new recipes. It was developed in Android Studio using Java and XML. Utilizes Food2Fork API to fetch recipe data. Java's 'AsyncTask' handles API responses, using URI reference for communication with server. Adapter classes populate nested views to fill list items. The demo page was created in HTML, JavaScript, jQuery. Developed in collaboration with Kathy Luo and Michael Mach.
